        Qual problema este código apresentará ao ser executado?

        D. O código não imprimirá nada, pois a condição numero < 3 é falsa desde o início.

        B. O código causará um erro de sintaxe porque while True é uma expressão inválida.
        C. Este é um loop infinito que imprimirá números indefinidamente, pois não há uma condição interna ou comando break para terminá-lo. (CORRETA)
        D. O loop terminará automaticamente quando contador atingir o limite máximo de um inteiro.

8. Combinar laços while com estruturas condicionais (if, elif, else) é uma prática recomendada. Qual é um dos principais benefícios dessa combinação, conforme a Aula 03?
        A. Simplificar a sintaxe básica do laço while para reduzir o número de linhas de código.
        B. Eliminar a necessidade de uma condição de término explícita no laço while.
        C. Permite criar programas mais robustos e dinâmicos, executando blocos de código específicos com base em condições verificadas a cada iteração do laço. (CORRETA)
        D. Acelerar o processamento de dados, transformando o while em um laço paralelo.

9. Considere o seguinte código com laços aninhados:
        i = 0
        while i < 2:
            j = 0
            while j < 2:
                print(f"Laço Interno: ({i},{j})")
                j += 1
            i += 1

        Quantas vezes a mensagem "Laço Interno: (i,j)" será impressa na tela?
        A. 2 vezes
        B. 3 vezes
        C. 4 vezes (CORRETA)
        D. 6 vezes

10. Analise o código que tenta encontrar um número par em um determinado intervalo e para a execução quando encontra:
        num = 7
        limite = 15
        while num <= limite:
            if num % 2 == 0:
                print(f"Número par encontrado: {num}")
                break
            num += 1
        else:
            print("Nenhum número par encontrado no intervalo.")

        Qual será a saída final deste código?
        A. Nenhum número par encontrado no intervalo.
        B. Número par encontrado: 8 (CORRETA)
        C. Número par encontrado: 14
        D. O código entrará em loop infinito.
