2.  Analise o seguinte trecho de código:
        const numeros = [10, 5, 0, -3, 8];
        let resultado = [];
        numeros.forEach(function(numero) {
            if (numero > 0) {
                resultado.push("Positivo");
            } else if (numero < 0) {
                resultado.push("Negativo");
            } else {
                resultado.push("Zero");
            }
        });
        console.log(resultado);
        Qual será a saída exata no console ao executar este script, de acordo com o uso do método forEach()?
        A. ["Positivo", "Positivo", "Zero", "Negativo", "Positivo"] (CORRETA)
        B. [10, 5, 0, -3, 8]
        C. undefined
        D. Erro: forEach não pode modificar arrays externos.

        console.log(todosAprovados);
